This chapter contains material for designing and developing applications with Mule.

Regardless of the integration development environment you choose to work with Mule – xref:5@studio::index.adoc[Anypoint Studio], Eclipse, IntelliJ, etc. – these materials aim to provide the details you need to configure the components of your application. Browse the child pages in this chapter to access documentation for both Mule ESB and Studio.
